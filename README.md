## ü§ñ Machine Learning & Deep Learning Projects

### 1. Artificial Neural Network (ANN) without PyTorch
- **Location**: `ANN_without_pytorch_n_all.ipynb`
- **Description**: Implementation of Artificial Neural Networks from scratch using only NumPy and mathematical operations, without relying on deep learning frameworks like PyTorch or TensorFlow.
- **Tech Stack**: Python, NumPy, Pandas, Matplotlib
- **Key Concepts**: Forward propagation, backpropagation, gradient descent, activation functions

### 2. Convolutional Neural Network (CNN) using Only Maths and NumPy
- **Location**: `CNN_using_only_maths_and_numpy(no_pytorch_TFs_Keras).ipynb`
- **Description**: Complete CNN implementation from scratch using pure mathematical operations and NumPy, demonstrating deep understanding of convolutional operations.
- **Tech Stack**: Python, NumPy, Matplotlib
- **Key Concepts**: Convolution, pooling, feature maps, CNN architecture

### 3. Image Classification (CIFAR-10 Dataset)
- **Location**: `Image_Classification_(CIFAR_10_Dataset).ipynb`
- **Description**: Image classification model trained on the CIFAR-10 dataset, exploring various architectures and optimization techniques.
- **Tech Stack**: Python, TensorFlow/Keras, NumPy, Matplotlib
- **Dataset**: CIFAR-10 (60,000 32x32 color images in 10 classes)

### 4. Rock vs Mine Prediction
- **Location**: `Rock_vs_Mine_Prediction.ipynb`
- **Description**: Binary classification model to predict whether an underwater object is a rock or a mine using sonar data.
- **Tech Stack**: Python, Scikit-learn, Pandas, NumPy
- **Application**: Sonar signal processing, underwater object detection

### 5. Titanic Survival Prediction with TensorFlow Decision Forests
- **Location**: `Titanic Prediciton with TF Decision Forests/`
- **Description**: Predicting passenger survival on the Titanic using TensorFlow Decision Forests, exploring gradient boosted trees and random forests.
- **Tech Stack**: Python, TensorFlow Decision Forests, Pandas, NumPy
- **Dataset**: Titanic dataset
